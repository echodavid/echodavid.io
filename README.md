# Clinical Patient Questionnaire

This project is a multi-page HTML form designed for a clinical patient questionnaire. It allows users to navigate through various sections of the questionnaire, collecting essential information about the patient's health and medical history.

## Project Structure

The project consists of the following files:

- **index.html**: The main entry point for the questionnaire. It includes links to the CSS and JavaScript files and sets up the basic structure of the HTML document.
  
- **css/styles.css**: Contains styles for the questionnaire, defining the layout, colors, fonts, and other visual aspects to enhance user experience.
  
- **js/script.js**: Includes JavaScript code that manages the form's functionality, such as navigation between pages, form validation, and dynamic question display based on user responses.
  
- **pages/page1.html**: The first part of the questionnaire, including questions about age, date of birth, sex, and related questions.
  
- **pages/page2.html**: The second part of the questionnaire, focusing on women's health questions, pregnancy history, and family medical history.
  
- **pages/page3.html**: The final part of the questionnaire, addressing lifestyle, vaccination history, and sexual health questions.

## Usage Instructions

1. Open `index.html` in a web browser to start the questionnaire.
2. Navigate through the pages using the provided buttons.
3. Fill out the questionnaire as prompted, ensuring all required fields are completed.
4. Submit the questionnaire at the end of the final page.

## Development

To modify the styles, edit the `css/styles.css` file. For any changes in functionality or form behavior, update the `js/script.js` file. Each page of the questionnaire can be customized by editing the respective HTML files in the `pages` directory.
